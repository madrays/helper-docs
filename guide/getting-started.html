快速开始 ​

本文将帮助你快速了解和开始使用 Sun-Panel-Helper。

在线体验 ​

在部署之前，你可以通过我们提供的演示环境体验完整功能：

  1. 访问 Helper 演示站

    • 地址：demo.cocoyoo.cn
    • 账号：helper
    • 密码：helper123

  2. 访问 Sun-Panel 演示站

    • 地址：home.cocoyoo.cn
    • 用于查看美化效果

使用建议

建议同时打开两个站点，这样可以边修改边查看效果，获得最佳体验！

基本要求 ​

在开始使用 Sun-Panel-Helper 之前，请确保：

  1. 已经安装并正常运行 Sun-Panel
  2. 已经安装 Docker 环境
  3. 可以访问 Sun-Panel 的配置目录

快速部署 ​

这里提供最基本的部署命令，详细部署说明请查看部署指南。

bash
# 创建Helper所需的数据目录（这些是Helper自己的目录，不是Sun-Panel的目录）
mkdir -p /your/helper/path/data
mkdir -p /your/helper/path/backups

# 启动容器
docker run -d \
  --name sun-panel-helper \
  -p 33002:80 \
  -e BACKEND_PORT=3001 \
  # Helper自己的数据目录（需要您自己创建）
  -v /your/helper/path/data:/app/backend/data \
  # Helper自己的备份目录（需要您自己创建）
  -v /your/helper/path/backups:/app/backend/backups \
  # 必须指向Sun-Panel的custom目录
  -v /path/to/sunpanel/conf/custom:/app/backend/custom \
  madrays/sun-panel-helper:latest

目录说明

  • /your/helper/path/data 和 /your/helper/path/backups 是您为Helper创建的数据目录，不是Sun-Panel的目录
  • /path/to/sunpanel/conf/custom 必须替换为您的Sun-Panel实际custom目录的绝对路径

访问面板 ​

  1. 打开浏览器访问 http://your-ip:33002
  2. 使用以下默认账号登录：
    • 账号：helper
    • 密码：helper123
